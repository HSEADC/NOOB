Сервисы для генерации
цветовых палитр

Полезные онлайн-инструменты для подбора гармоничных цветовых сочетаний

Coolors

Один из самых популярных генераторов цветовых схем. Можно генерировать случайные палитры, сохранять свои, экспортировать в разных форматах. Очень удобен для быстрых экспериментов.

Перейти
ColorSpace

Инструмент, который создаёт палитры на основе одного выбранного цвета. Генерирует сразу множество вариантов: градиенты, оттенки, фоновые сочетания. Подходит для UI, презентаций и брендинга.

Перейти
Khroma

Нейросеть, которая подбирает палитры на основе твоих цветовых предпочтений. Нужно отметить любимые цвета, и алгоритм предложит вдохновляющие комбинации.

Перейти
Adobe Color

Профессиональный инструмент от Adobe. Можно строить палитры по правилам цветового круга, загружать изображения для извлечения палитры и сохранять в Creative Cloud.

Перейти
Paletton

Инструмент для точной настройки палитр по цветовым гармониям: монохромной, аналоговой, комплементарной и другим. Полезен для тех, кто работает с теорией цвета.

Перейти
Color Hunt

Коллекция стильных цветовых палитр от дизайнеров со всего мира. Удобно листать, выбирать и сохранять понравившиеся варианты. Обновляется ежедневно.
