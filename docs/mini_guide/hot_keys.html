Горячие клавиши Figma

Все полезные горячие клавиши для Figma

Основные инструменты для Windows / Mac

Move Tool (Перемещение) — V

Scale Tool (Масштаб) — K

Frame (Артборд) — F

Rectangle (Прямоугольник) — R

Ellipse (Овал) — O

Line (Линия) — L

Arrow (Стрелка) — Shift + L

Pen Tool (Перо) — P

Hand Tool (Рука) — Space

Comment Tool (Комментарий) — C

Работа с объектами

Группировать — Ctrl + G / Cmd + G

Разгруппировать — Ctrl + Shift + G / Cmd + Shift + G

Дублировать — Ctrl + D / Cmd + D

Переименовать — Ctrl + R / Cmd + R

Заблокировать объект — Ctrl + Shift + L / Cmd + Shift + L

Скрыть объект — Shift + Ctrl + H / Shift + Cmd + H

Выравнивание — Alt + A, Alt + H, Alt + V, и т.п. (по контексту)

Автолейаут — Shift + A

Цвет, текст и стили

Открыть палитру заливки — Shift + B

Копировать стиль — Ctrl + Alt + C / Cmd + Option + C

Вставить стиль — Ctrl + Alt + V / Cmd + Option + V

Создать компонент — Ctrl + Alt + K / Cmd + Option + K

Отобразить/скрыть границы фреймов — Ctrl + Shift + 4 / Cmd + Shift + 4

Просмотр и навигация

Зум 100% — Ctrl + 1 / Cmd + 1

Подогнать к экрану — Shift + 1

Показать пиксельную сетку — Ctrl + ' / Cmd + '

Переключить тёмную/светлую тему — Ctrl + \ / Cmd + \

Навигация и панель

Показать/скрыть UI — Ctrl + \ / Cmd + \

Поиск (быстрые команды) — Ctrl + / / Cmd + /

Перейти к слою — Ctrl + Click (на холсте)

Навигация по слоям — Tab / Shift + Tab
